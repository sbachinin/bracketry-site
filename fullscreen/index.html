PLAYOFFSjs
Visualize a knockout tournament in the browser
Fullscreen mode
Open
How

Set fullscreen option to true:

import { createPlayoffs } from 'playoffs'

const options = {
    fullscreen: true,
    width: '90vw',
    height: '90vh'
}

document.querySelector('.your-button').addEventListener('click', () => {
    createPlayoffs(yourData, document.body, options)
})
Size

By default Playoffs content will take the entire viewport. Though you may want to have some nice paddings on the edges of viewport. There is no such option as fullscreen padding but there are good old width and height options which may be set to some vw or vh value or something with calc. These paddings will be filled with fullscreenBgColor. Actually this color will fill the entire viewport if no rootBgColor option is provided.

"Mixed" vertical scroll mode will be forced

verticalScrollMode option will be automatically set to "mixed" when "fullscreen" is true. Setting verticalScrollMode to any other value will not work here. Vertical scroll buttons will be shown and mousewheel will also scroll the matches. On mobile screens only buttons will work; touchmove is disabled in fullscreen mode.

Ways to exit fullscreen mode

   - click the cross button    - click anywhere outside the playoffs' "body"    - press Esc    - (programmatically) call uninstall method on a playoffs' instance

"fullscreen" option is not updatable

This option will be ignored when passed to applyNewOptions. It means you can't switch between "fullscreen" and "static" mode within one instance of playoffs. If you need "small" playoffs which are expandable to fullscreen, then you need 2 instances, one "static" and one "fullscreen" which should be created in response to some ui events (like button click. But you have to make this button yourself. So yeah, it's difficult).
