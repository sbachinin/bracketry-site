How to apply live updates

applyMatchesUpdates()

It's one of the functions returned from createPlayoffs()
You can call it to add / remove / update any number of matches.

Parameters
  • Matches: an array of match objects that you want to update
Return value
    undefined

Every match object provided to applyMatchesUpdates must contain ALL data of a match
(It's not a merge but replace. If you omit some properties, on the next paint they will be treated as undefined) 

import { createPlayoffs } from 'playoffs'

const wrapper = document.querySelector('#your-wrapper-element')
const data = { ... }

const playoffs = createPlayoffs(data, wrapper)

playoffs.applyMatchesUpdates([
    {
        roundIndex: 0,
        order: 0,
        ...any match data
    }
])
