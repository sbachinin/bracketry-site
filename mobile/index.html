Provide flexible styles

To make playoffs shrink on smaller screens you have to set appropriate styles.
First of all, make sure that your wrapper element (that in which playoffs are installed) has a 'fluid' width, i.e. it has for example a style of { max-width: 100vw } or something of this sort.
Width can also be set on Playoffs via width option but still it makes more sense to set width on your wrapper (and playoffs by default will take 100% of its width).
