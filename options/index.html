How to apply options
Options can be passed to createPlayoffs as a 3rd argument 
import { createPlayoffs } from 'playoffs'

const wrapper = document.querySelector('#your-wrapper-element')
const data = { ... } // data of specific shape

const options = {
    matchMaxWidth: '200px'
}

createPlayoffs(data, wrapper, options)

Options can be changed later via applyNewOptions lifecycle method 
import { createPlayoffs } from 'playoffs'

const wrapper = document.querySelector('#your-wrapper-element')
const data = { ... } // data of specific shape

const options = {
    matchMaxWidth: '200px'
}

const { applyNewOptions } = createPlayoffs(data, wrapper, options)

applyNewOptions({
    matchMaxWidth: '100px'
})
Options passed to applyNewOptions will be merged with those which were passed to createPlayoffs()
Options of functional type (such as getMatchElement) are not changeable by applyNewOptions

You can get current options via getUserOptions method
