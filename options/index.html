How to set options on installation

Options can be passed to createPlayoffs as a 3rd argument

import { createPlayoffs } from 'playoffs'

const wrapper = document.querySelector('#your-wrapper-element')
const data = { ... } // data of specific shape

const options = {
    matchMaxWidth: '200px'
}

createPlayoffs(data, wrapper, options)
How to set options later

Options can be changed later via applyNewOptions lifecycle method

...

const { applyNewOptions } = createPlayoffs(data, wrapper, options)

applyNewOptions({
    matchMaxWidth: '100px'
})

Options passed to applyNewOptions will not replace the original ones (passed to createPlayoffs) but will be merged.
Some options are not updatable (ignored when passed to applyNewOptions):
    - "fullscreen"
    - "verticalScrollMode"
    - options of type "function" (such as "getMatchElement")
The most likely case for applyNewOptions is handling different screen sizes. (You may respond to resize by setting more "compact" playoffs' layout)

Use getUserOptions method to get current options.
