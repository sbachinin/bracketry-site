How to change a tournament (replace all data)

replaceData()

It's one of the functions returned from createPlayoffs() Calling this method is almost identical to installing playoffs (calling createPlayoffs) into the same wrapper element.

Parameters
  • Data: full data of a tournament
Return value
    undefined 

import { createPlayoffs } from 'playoffs'

const wrapper = document.querySelector('#your-wrapper-element')
const data = { ... }

const playoffs = createPlayoffs(data, wrapper)

playoffs.replaceData(someNewData)

If you want to update individual matches (e.g., only some scores), replaceData will NOT be the best idea. Use applyMatchesUpdates instead.
